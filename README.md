# retro-game-linux
Trabalho de emulação de jogos retro no linux

Jandira Pires

 1. Baixar e instalar programa Virtual Box
 2. Criar VM
 3. Baixar Linux
 4. Inicializar VM
 5. Selecionar 

> Dispositivos -> Disco óptico -> Imagem disco ->Linux_Mint .iso

 6. Reinicializar VM
 7. Instalar Linux Mint na VM
 8. Atualizar Linux Mint na VM
 9. Escolher o emulador de games
 10. Abrir o terminal
 11. Instalar emulador escolhido
 12. Fechar terminal
 13. Baixar rom 
 14. Abrir emulador
 15. Selecionar: 

> Game -> Load-> Jogo

  
 16. Abrir 

> pasta pessoal usuário -> documentos -> abrir terminal

 17. Rodar o comando 

> sudo apt-get install git
18. Confirmar senha
19. Confirmar instalação
20. Criar repositório no GitHub
21. No terminal 
> git clone URL_do_repositorio
> ls --> visualiza o que tem na pasta
22. Abrir pasta do repositório
23. Alterações
24. Rodar o comando 
> git add - A
> git commit -m "msg alteração"
> git push origin master
25. Fim do processo.
